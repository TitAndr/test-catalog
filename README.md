# Product Catalog (Test Task)

Реалізація тестового завдання на базі TypeScript + React + Vite + Tailwind CSS.

## Що реалізовано

- Отримання товарів з API:
  - `https://ip-194-99-21-145-139178.vps.hosted-by-mvps.net/api/v1/products`
- Відображення каталогу товарів картками
- Пошук за назвою товару
- Сортування:
  - ціна за зростанням
  - ціна за спаданням
  - рейтинг
  - знижка
  - назва
- Пагінація
- Обробка станів завантаження та помилки

## Запуск локально

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```
